### 基于ChatGPT Web docker-compose 部署教程
- 在原来基础上增加会员注册，登录功能,需要后台管理自行用dcat-admin写一个
- ```shell
  # 在docker-compose目录下env.dev配置好OPENAI_API_KEY
  # 初始化
  sh ./script/init.sh
  ```
- ```shell
  # 启动
  sh ./script/start.sh
  ```
有任何不明白加V
